=== FOLDER GAMBAR SAMPEL ===

Folder ini berisi beberapa gambar sampel untuk pengujian RJ Auto Metadata.

Tujuan gambar sampel:
1. Membantu Anda menguji aplikasi tanpa menggunakan file produksi
2. Menunjukkan berbagai format dan ukuran file yang didukung
3. Memberikan contoh bagaimana metadata akan ditambahkan ke gambar

Daftar gambar sampel:
- sample1.jpg - Foto lanskap (2.07 MB)
- sample2.jepg - Foto portrait (1.59 MB)
- sample3.png - Gambar dengan latar belakang transparan (7.97 MB)
- sample4.jpg - Gambar besar yang akan dikompresi otomatis (20.9 MB)

Cara menggunakan gambar sampel:
1. Salin gambar-gambar ini ke folder terpisah untuk pengujian
2. Pilih folder tersebut sebagai folder input di aplikasi
3. Pilih lokasi output yang berbeda
4. Jalankan proses untuk melihat bagaimana aplikasi bekerja

PENTING: Gambar-gambar sampel ini hanya untuk tujuan pengujian dan pelatihan.
Semua gambar bebas hak cipta dan dapat digunakan untuk pengujian non-komersial.
